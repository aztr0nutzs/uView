# Screen Authority Map

## Authoritative Root HTML Files

- `uview_screen1.html` -> dashboard / camera list / tactical hub
- `uview_screen2.html` -> multi-view tactical grid
- `uview_screen3.html` -> camera detail / live feed

## Mandatory Rules

- These three HTML files are not inspiration boards. They are the visual source of truth.
- Their exact colorway, layout language, spacing density, typography feel, edge treatment, and HUD framing must be preserved.
- Redesign is allowed only when needed to complete missing states or improve usability without changing the visual identity.
- No phase may simplify the UI into generic Material design patterns.
